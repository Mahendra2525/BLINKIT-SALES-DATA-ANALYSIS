### Blinkit Sales Data Analysis
#### Project Description
This project analyzes sales data from Blinkit (India’s last-minute delivery app) using an interactive dashboard. 
It provides insights into sales performance,customer preferences, outlet distribution, and product trends.
Dashboard Overview
#### 1️⃣ Summary Dashboard (First Image)

Shows:
KPIs (Sales, Items, Ratings)
Sales by outlet size & location
Product category performance
Trends over time

#### 2️⃣ Data Table View (Second Image)

Shows:

Item-level data (Item Identifier, Weight, Visibility)

Outlet details (Type, Size, Location)

Ratings and establishment year

👉 This combination allows both high-level insights + detailed analysis

#### 🔢 Key Metrics

Total Sales: $1.20M

Total Items Sold: 9K

Average Sales: 140.99

Average Rating: 3.92

📊 Detailed Analysis
🏪 Outlet Performance

Supermarket Type1 → highest revenue contributor

Large outlets generate more sales than small ones

👉 Business depends heavily on large supermarket formats

#### 📍 Location Insights

Tier 3 cities generate the highest sales

Tier 2 and Tier 1 follow behind

👉 Strong demand in non-metro cities

#### 🧈 Product Preferences

Regular fat products sell more than low-fat

Top categories:

Fruits & Snacks

Household

Dairy

👉 Customers prefer daily essentials over niche products

#### 📅 Time-Based Trends

Sales increased with outlet expansion

Peak observed around $205K sales period

👉 Growth driven by new outlet openings

#### 📋 Data Table Insights (Second Image)

The table provides:

Item-level tracking (e.g., DRA12, DRA24, etc.)

Visibility and weight impact on sales

Outlet segmentation:

Type (Supermarket, Grocery Store)

Size (Small/Medium/High)

Location (Tier 1/2/3)

👉 Helps in:

Identifying underperforming products

Comparing outlet efficiency

Deep-dive analysis for decision making

#### 🧠 Key Business Insights

Tier 3 cities are the strongest revenue drivers

Supermarket Type1 dominates sales

Larger outlets outperform smaller ones

Regular products are more popular than low-fat

Business growth is linked to outlet expansion

#### 🚀 Business Recommendations

Expand more outlets in Tier 3 cities

Focus on Supermarket Type1 model

Increase stock of high-demand categories

Improve marketing for low-performing products

Optimize outlet size for better revenue
